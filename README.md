# spotwo
우리가 함께 채운 지도 💗

---
Phase 1 - 기초    
├── Docker Compose 세팅       
├── Entity 설계 & JPA     
├── JWT 인증 (회원가입/로그인)       
└── 커플 매칭 (초대코드)

Phase 2 - 핵심 기능    
├── 기록 CRUD (사진 + 글)        
├── MinIO 파일 업로드        
├── 지도 핀 API        
└── 계절/날씨 자동태깅

Phase 3 - 프론트       
├── React + Kakao Map 연동        
├── PWA 설정 (모바일 앱처럼)        
└── 반응형 UI

Phase 4 - 부가 기능     
├── 위시리스트 핀     
├── 지역 통계       
└── 푸시 알림
